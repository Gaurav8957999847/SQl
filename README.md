# SQL Practice (MySQL)

This repository contains my SQL practice queries written using MySQL.
The goal of this repository is to practice core SQL concepts such as filtering, joins, grouping, and subqueries.

## Topics Covered

* Basic Queries
* Filtering with WHERE
* Sorting with ORDER BY
* Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)
* GROUP BY and HAVING
* Joins
* Subqueries

## Repository Structure

```
sql-practice
│
├── basics.sql        -- simple SELECT queries
├── filtering.sql     -- WHERE conditions
├── joins.sql         -- INNER JOIN, LEFT JOIN
├── groupby.sql       -- GROUP BY and HAVING
├── subqueries.sql    -- nested queries
```

## Database Used

MySQL

## Purpose

This repository is part of my SQL learning process.
Each file contains queries written while practicing different SQL concepts.

## How to Use

1. Clone the repository.
2. Open the `.sql` files in MySQL Workbench.
3. Run the queries on your local database to practice.
